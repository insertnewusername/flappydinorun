# flappydinorun
<h1>FlappyDino game</h1>

FlappyDino game is flappy bird, and chrome dino game. All in ONE! With only onekey needed, there can't realy be simpler games to learn how to play.


<h1>Yap from me</h1>
After my previous Godot project, I realised that most of it was simply playing around with the tilemap system.
In this project, I wanted to work with godot code more instead of mere tiles. Which is why I decided to recreate flappy bird and
google chrome dino game. Originally, I was also planning to add geometry dash, but I realised that it didn't really suit the theme.

Making this project introduced several new components of Godot, such as more scripting features, such as going back to start, selecting a random costume, sound effects, etc.
